# Field Reference Names for Bug Triage (CLE Project)

This document lists the Azure DevOps field reference names identified from the triage query results for the 'CLE' project. Use these names when interacting with the `ado-update` tool.

*   **ID**: `System.Id`
*   **Title**: `System.Title`
*   **State**: `System.State`
*   **Assigned To**: `System.AssignedTo`
*   **Priority**: `Microsoft.VSTS.Common.Priority`
*   **Iteration Path**: `System.IterationPath`
*   **Created Date**: `System.CreatedDate`
*   **Description**: `System.Description`
*   **Repro Steps**: `Microsoft.VSTS.TCM.ReproSteps`
*   **Comment/History**: `System.History` (Used for adding comments via `ado-update`)
*   **Triage State**: `Office.Triage.TriageState` (**Note**: This field may only exist in the `office.visualstudio.com` instance.)
