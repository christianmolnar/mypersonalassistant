# Communication Templates

This document provides templates for common communication types including emails, announcements, executive summaries, and customer communications.

## Email Templates

### Project Update Email

```
Subject: [Project Name] Update: [Key Milestone/Status]

Hi [Team/Name],

I wanted to share a quick update on the [Project Name] progress:

**What's been completed:**
- [Achievement 1]
- [Achievement 2]
- [Achievement 3]

**Current focus:**
- [Current priority 1]
- [Current priority 2]

**Timeline update:**
We're [on track/ahead of schedule/experiencing delays] for our [target date]. [Additional context if needed]

**Action needed:**
- [Action item 1] by [date]
- [Action item 2] by [date]

Please let me know if you have any questions or need any additional information.

Thanks,
[Your name]
```

### Request for Input/Feedback

```
Subject: Need your input: [Brief Topic] by [Deadline]

Hi [Name],

I'm working on [brief description of project/task] and would value your input on [specific area].

**Background:**
[1-2 sentences providing necessary context]

**Specifically, I'd like your thoughts on:**
- [Question or area for input 1]
- [Question or area for input 2]
- [Question or area for input 3]

**Timeline:**
Could you please share your feedback by [date]? This will help us [benefit of meeting the timeline].

**How to provide feedback:**
[Specify method - reply to email, comment on document, schedule meeting, etc.]

Thanks in advance for your help!

[Your name]
```

## Team Announcement Templates

### New Initiative Announcement

```
Subject: Introducing [Initiative Name]: [Brief value proposition]

Hi team,

I'm excited to announce [Initiative Name], a new effort to [brief description of purpose and value].

**What it is:**
[2-3 sentences describing the initiative]

**Why we're doing this:**
[1-2 sentences explaining the strategic importance or benefits]

**How it impacts you:**
[Specific impacts on the team, their work, or processes]

**Timeline and next steps:**
- [Key date/milestone 1]: [Description]
- [Key date/milestone 2]: [Description]
- [Key date/milestone 3]: [Description]

**Learn more:**
- [Link to documentation, if available]
- [Information about upcoming info sessions]
- [Contact person for questions]

Please [specific call-to-action] by [deadline if applicable].

Thanks,
[Your name]
```

## Customer Communication Templates

### Product Update Notification

```
Subject: Important Update to [Product Name]

Dear [Customer Name],

We're reaching out to let you know about an upcoming update to [Product Name] that will [brief benefit statement].

**What's changing:**
- [Change 1]: [Brief description and benefit]
- [Change 2]: [Brief description and benefit]
- [Change 3]: [Brief description and benefit]

**When this happens:**
This update will be released on [date] at approximately [time with timezone].

**What you need to do:**
[Clear instructions for any customer action needed, or "No action is required on your part."]

**Resources to help:**
- [Link to documentation]
- [Link to FAQ]
- [Link to support resources]

If you have any questions, please [contact instructions].

Thank you for being a valued customer.

Regards,
[Your name/team name]
[Company name]
```

### Support Response Template

```
Subject: RE: [Original Subject Line] [Case/Ticket #]

Hello [Customer Name],

Thank you for contacting us about [brief restatement of their issue]. I understand how [acknowledge impact/frustration] this situation is.

**What I understand:**
[Restate the customer's issue to show understanding]

**Current status:**
[Information about what has been done to investigate/resolve so far]

**Next steps:**
[Clear information about what will happen next]
- [Step 1 with timing if applicable]
- [Step 2 with timing if applicable]

[One of the following:]
I've resolved this by [solution implemented] and you should now be able to [expected outcome].

OR

I'm still working on this and will update you by [specific time/date].

**To prevent this in the future:**
[If applicable, tips to avoid the issue or information about permanent fixes planned]

Please let me know if you have any questions or if there's anything else I can help with.

Best regards,
[Your name]
[Role/Team]
[Contact information]
```
